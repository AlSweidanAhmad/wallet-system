# wallet-system
A simple Java-based wallet system implementing clean code principles, OOP and reliable monetary transactions.

## Feautres
- Deposit and withdrawal operations 
- Transactions handling
- Precise money calculations using BigDecimal
- Clean and structured codebase

## Technologies
- Java
- OOP
- BigDecimal
- Clean Code

## Projekt Structure
- Domian-focused design
- Clearseparation of responsibilities

- Open the Project in your IDE
- Run the main class
